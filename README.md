🔥 2. Par ou Ímpar

Peça ao usuário que digite um número.
O programa deve verificar se o número é par ou ímpar e mostrar o resultado.

🔥 3. Calculadora Simples

Peça ao usuário dois números e, em seguida, peça que ele escolha uma operação matemática (+, -, *, /).
O sistema deve realizar a operação escolhida e exibir o resultado.

🔥 4. Tabuada Personalizada

Peça ao usuário um número.
O programa deve mostrar a tabuada desse número de 1 a 10.

🔥 5. Média do Aluno

Peça ao usuário três notas.
Calcule a média e informe se ele está:

Aprovado (média maior ou igual a 7)

Recuperação (média maior ou igual a 5 e menor que 7)

Reprovado (média menor que 5)

🔥 6. Contagem até N

Peça ao usuário um número inteiro positivo.
O programa deve exibir todos os números de 1 até o número digitado.

🔥 7. Soma até Digitar Zero

Peça números ao usuário repetidamente.
O programa deve continuar pedindo números até que o usuário digite 0.
Ao final, deve mostrar a soma total dos números digitados.

🔥 8. Contador de Números Positivos

Peça ao usuário que digite 5 números.
O programa deve informar quantos desses números são positivos.

🔥 9. Sistema de Login

Crie um sistema simples de login.
O programa deve pedir usuário e senha.

Usuário correto: "admin"

Senha correta: "1234"

Permita até 3 tentativas.
Se acertar, exiba mensagem de sucesso.
Se errar as 3 vezes, informe que o acesso foi bloqueado.

🔥 10. Cálculo de Fatorial

Peça ao usuário um número inteiro positivo.
O programa deve calcular e mostrar o fatorial desse número.
